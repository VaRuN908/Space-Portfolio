# Space Portfolio

This is a personal portfolio website built with Next.js, Tailwind CSS, and Framer Motion. It showcases my projects and skills in a visually appealing space-themed design.

## Features

- **Responsive Design:** The website is fully responsive and works on all screen sizes.
- **Animations:** The website uses Framer Motion to create smooth and engaging animations.
- **3D Graphics:** The website uses Three.js and React Three Fiber to create a 3D star background.
- **Sections:** The website has three main sections:
  - **About Me:** A brief introduction about me.
  - **Skills:** A list of my skills and technologies I've worked with.
  - **Projects:** a showcase of my projects with links to the source code and live demos.

## Technologies Used

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Three.js](https://threejs.org/)
- [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction)

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the development server
   ```sh
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
